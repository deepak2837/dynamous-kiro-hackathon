# StudyBuddy System Prompts

This folder contains various system prompts used throughout the StudyBuddy application for different AI-powered features.

## Prompt Categories

### Content Generation
- `content_analysis.txt` - For analyzing uploaded study materials
- `question_generation.txt` - For generating MCQs from content
- `mock_test_creation.txt` - For creating comprehensive mock tests
- `mnemonic_generation.txt` - For creating memory aids and mnemonics
- `cheat_sheet_creation.txt` - For summarizing key points
- `note_taking.txt` - For generating structured notes

### Medical/Educational Specific
- `medical_content_analysis.txt` - Specialized for medical content
- `indian_medical_context.txt` - India-specific medical education prompts
- `difficulty_classification.txt` - For classifying question difficulty

### Content Processing
- `text_extraction.txt` - For OCR and text processing
- `image_analysis.txt` - For analyzing images and diagrams
- `video_analysis.txt` - For processing video content

### User Interaction
- `explanation_generation.txt` - For creating detailed explanations
- `feedback_generation.txt` - For providing study feedback
- `progress_analysis.txt` - For analyzing learning progress

## Usage

These prompts are loaded dynamically by the application based on the requested functionality. Each prompt is optimized for specific use cases and can be customized as needed.

## ðŸƒ Flashcard System

### Spaced Repetition Learning

#### How It Works
The flashcard system uses the **SM-2 algorithm** optimized for medical education:

1. **Initial Learning**: New cards shown frequently
2. **Difficulty Rating**: Rate your understanding (1-5)
3. **Adaptive Scheduling**: Next review based on performance
4. **Long-term Retention**: Gradually increasing intervals

#### Review Schedule
```
Rating 5 (Easy): Next review in 4 days
Rating 4 (Good): Next review in 2 days  
Rating 3 (Medium): Next review in 1 day
Rating 2 (Hard): Next review in 1 hour
Rating 1 (Again): Immediate review
```

### Using Flashcards

#### Starting Flashcard Review
1. Go to **Flashcards** tab in results viewer
2. Click **"Start Review Session"**
3. Choose review type:
   - **New Cards**: Learn new content
   - **Due Cards**: Review scheduled cards
   - **All Cards**: Complete review session

#### Review Interface
1. **Question Side**: Read the medical question/concept
2. **Think**: Formulate your answer mentally
3. **Reveal Answer**: Click to see the correct answer
4. **Rate Difficulty**: Choose 1-5 based on how well you knew it
5. **Next Card**: Continue to next flashcard

#### Flashcard Categories
- **Anatomy**: Structural relationships and clinical correlations
- **Physiology**: Functional mechanisms and applications
- **Pathology**: Disease processes and diagnostic features
- **Pharmacology**: Drug mechanisms and clinical usage
- **Clinical**: Case-based scenarios and management

### Progress Tracking

#### Review Statistics

- **Accuracy Rate**: Percentage of correct responses

- **Time Spent**: Total study time with flashcards

#### Learning Analytics
- **Retention Rate**: Long-term memory performance

## ðŸ”§ Troubleshooting

### Common Issues and Solutions

#### Upload Issues

**Problem**: Files not uploading
```
Solutions:
1. Check file size (PDF: 50MB, Images: 10MB, PPTX: 100MB)
2. Verify file format (PDF, JPG, PNG, PPTX only)
3. Check internet connection stability
4. Try uploading one file at a time
5. Clear browser cache and cookies
```

**Problem**: Processing takes too long
```
Solutions:
1. Large files take 5-10 minutes (normal)
2. Check processing status regularly
3. Don't close browser during processing
4. Try uploading smaller files
5. Contact support if stuck >15 minutes
```

#### Content Quality Issues

**Problem**: Generated questions are not relevant
```
Solutions:
1. Upload more specific medical content
2. Use clear, well-structured study materials
3. Include medical terminology in uploads
4. Try topic-based generation instead
5. Provide feedback on question quality
```

**Problem**: Mnemonics are not helpful
```
Solutions:
1. Upload content with clear concepts
2. Include context and explanations
3. Try different types of study materials
4. Use topic input for specific concepts
5. Create custom mnemonics manually
```

#### Technical Issues

**Problem**: Login/authentication issues
```
Solutions:
1. Verify mobile number format (10 digits)
2. Check OTP in SMS/email (including spam)
3. Wait 2 minutes before requesting new OTP
4. Try different browser or incognito mode
5. Clear browser data and try again
```

**Problem**: Session not loading
```
Solutions:
1. Refresh the page
2. Check internet connection
3. Try different browser
4. Clear browser cache
5. Contact support with session ID
```

#### Performance Issues

**Problem**: App running slowly
```
Solutions:
1. Close other browser tabs
2. Check internet speed
3. Try different browser
4. Clear browser cache
5. Restart browser/device
```

**Problem**: Export/download not working
```
Solutions:
1. Check browser download settings
2. Disable popup blockers
3. Try different file format
4. Check available storage space
5. Try downloading smaller content
```
